How to install Additional Providers

To install any provider on this package, we simply need to copy the provider folder content, and then merge it into the HybridAuth core library.

For the sake of the example, lets say you want to add Foursquare to your authetication service:

  1. First, open hybridauth-foursquare/ folder,
  2. Copy hybridauth-foursquare/Providers/ and hybridauth-foursquare/thirdparty/ folders,
  3. Then, past them into HybridAuth core library: hybridauth-2.x.x/Hybrid/Providers/ and hybridauth-2.x.x/Hybrid/thirdparty/,
  4. Goto Foursquare adapter specification web page on the online user guide to find out how to setup an new application,
  5. Finally, add the created foursquare application credentials to your configuration :
return 
    array( 
        "base_url" => "http://mywebsite/path/to/hybridauth/", 

        "providers" => array (
            // ... others providers already setup ... // 

            // foursquare application credentials
            "Foursquare" =>   array ( 
               "enabled"   => true,
               "keys"   => array ( "key" => "*****************", "secret" => "*********************************" )
            ) 
        ),
    );

Well, thats it. If you have any issue installing or using any providers drop me a line at hybridauth@gmail.com
You can also find the complete HybridAuth documentation at http://hybridauth.sourceforge.net/.
